
# Playing about with Three.js from Clojurescript 


Tools: 

* shadow
* UIx 
* react-three-fiber (use three.js from react)

## How to 

* start repl with Clojure CLI
* compile 'cljs' ns 
* start building the app `(cljs/watch-app false)`
* visit http://localhost:9000
